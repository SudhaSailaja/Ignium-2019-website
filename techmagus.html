Techmagus



Do you inhale code and exhale bugs? Meet more like you at TECHMAGUS. IGNIUM 2K18 presents to you TECHMAGUS, an intense coding competition where you get to unleash the versatile coder in you.

Rules



ELIGIBILITY:The event is open to all B.Tech students who are interested in Coding in C.
EVENT DESCRIPTION:
Round 1:
Improve your debugging skills :
The errors must be debugged and corrections are to be made to C code which consists of some logical and syntactic errors. The participants who will be able to execute the code correctly in minimum time will be selected for the next round.
Round 2:
Driving you out of your comfort zone :
To go to round 3, successful execution of the code for the given problem statement without using certain constructs and data structures as specified, must be done.
Round 3:
Challenging your logical ability :
Winners will be finalized based on their logical ability to determine the relationship between the inputs and outputs given and write successfully executable code with minimum time & space complexity.
